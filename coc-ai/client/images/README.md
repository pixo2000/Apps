# Game Images

This directory should contain the following image files for the game:

## Resource Images
- `gold.png` - Gold icon
- `elixir.png` - Elixir icon

## Building Images
- `townhall.png` - Town Hall building
- `goldmine.png` - Gold Mine building
- `elixircollector.png` - Elixir Collector building
- `cannon.png` - Cannon defense building

## Troop Images
- `barbarian.png` - Barbarian troop
- `archer.png` - Archer troop

You can create simple placeholder images or find free game assets online. Make sure to use images that are approximately 64x64 pixels for buildings and 32x32 pixels for resources and troops.
