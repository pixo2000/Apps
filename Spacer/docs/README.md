# Spacer Documentation

Welcome to the official documentation for Spacer, an interstellar exploration simulator.

## About Spacer

Spacer is a text-based space exploration game where you navigate through different dimensions, discover celestial bodies, and explore the cosmos. As a captain of your own vessel, you'll chart unknown territories, scan planetary systems, and record your findings.
